📖 操作说明
快速上手 · 核心操作

🎯 快速上手（5步）

  1. 加载音频 → 点击「📂 加载音频」选择文件，或拖拽音频文件到页面
  2. 选择模式 → 点击 LRC（歌词）或 SRT（字幕）
  3. 添加内容 → 双击波形或按 Enter 键，输入文字（SRT 模式会自动添加当前时间作为开始时间）
  4. 添加时间戳 → 点击「⏱ 更新时间戳」或按空格键（需在配置中勾选「空格键更新时间戳」）
  5. 导出文件 → 点击「💾 导出」，选择格式后保存

📝 两种模式

  • LRC 每行格式：[mm:ss.mmm]歌词内容，适合卡拉OK歌词
  • SRT 每行含 开始 和 结束 两个时间戳，适合视频字幕

⏱ 更新时间戳

  • LRC模式：依次为每行添加时间戳；若勾选「中英文歌词/字幕」，则同时为下一行设置相同时间戳
  • SRT模式：点击时间范围中的 开始 或 结束（高亮后）→ 点击按钮单独更新；无选中时按顺序自动填充开始或结束时间
  • 只更新结束时间（SRT模式专用）：勾选后，每次只更新当前行的结束时间，不改变开始时间；若同时勾选「中英文歌词/字幕」，则成对更新相邻两行的结束时间
  • 所有选项可在 ⚙️ 配置 弹窗中统一管理

📍 左右边界微调

  • 点击「⏪ 左边界」按钮，程序会检测当前位置左侧最近的有声段起始点，然后自动「加上」边界放宽值，让游标向右（进入有声区内部）偏移。
  • 点击「⏩ 右边界」按钮，程序会检测当前位置右侧最近的有声段结束点，然后自动「减去」边界放宽值，让游标向左（回到有声区内部）偏移。
  • 「边界放宽」值可在工具栏中调整（步长 50ms，默认 100ms），方便您根据音频特点（如是否需要包含句前/句后呼吸声）灵活微调。
  • 如果未检测到有效边界，则直接在当前游标位置向右或左移动对应的放宽值。

🔄 时间戳转移

  • 当您在 LRC 模式 下已有带时间戳的歌词时，点击「时间戳转移」按钮，可将每行歌词的时间戳复制为 SRT 模式的 开始时间，并自动切换到 SRT 模式。
  • 此操作不会删除原 LRC 数据，转移后您可继续为 SRT 字幕添加结束时间。
  • 若 LRC 中存在未打时间戳的行，则对应 SRT 的开始时间会留空，您可后续单独补填。

📍 波形定位

  • 游标定位到波形：选中某行歌词/字幕（点击该行使其高亮），点击此按钮，波形播放头将跳转到该行的时间戳位置（LRC模式为 time，SRT模式为 start），方便您快速试听对应的音频片段。
  • 跳转最新时间戳：无论当前选中哪一行，点击此按钮会自动滚动到列表中最后一个拥有有效时间戳的行（忽略时间为 0 或空的行），并高亮该行，同时波形跳转到该时间位置。适合在处理大量歌词/字幕时快速定位到已处理的最新进度。

🗑️ 删除选中行

  • 勾选需要删除的行（支持多选），点击「🗑️ 删除选中行」按钮，确认后删除。
  • 删除后剩余行号自动重新排序，“最新时间戳”同步更新。
  • 支持 Ctrl+Z 撤销删除。

🧹 清除时间戳

  • 勾选需要清除时间戳的行，点击「🧹 清除时间戳」按钮，清空选中行的所有时间信息（LRC清空时间，SRT清空开始/结束）。
  • 不影响未选中的行，“最新时间戳”自动重算。

🔄 撤销 / 重做

  • 适用于 更新时间戳、清除时间戳、删除选中行 三类操作。
  • 最多保存最近 10 步，快捷键 Ctrl+Z 撤销，Ctrl+Y 重做。
  • 撤销/重做后，自动选中第一个未完成的时间戳行（LRC模式为 time 为空或0；SRT模式为 start 或 end 为空或0），并滚动到可视区域，方便您继续编辑。

▶️ 预览功能

  • 点击「▶ 预览」按钮，从当前游标位置开始播放，歌词/字幕行随播放进度自动高亮并滚动到可视区域。
  • 中英文歌词/字幕：若勾选「中英文歌词/字幕」复选框，预览时优先高亮第一行（通常为中文）；取消勾选则高亮最后一行（默认）。
  • 再次点击「预览」按钮可停止预览并暂停音频；播放结束后预览自动退出。
  • 预览过程中您仍可拖拽波形或点击列表项，高亮会跟随当前位置。
  • 提示：预览模式下点击「暂停」按钮会暂停播放，但预览高亮保持；如需完全停止预览请再次点击「预览」按钮。

↔️ 对齐切换

  • 工具栏中新增 「左对齐」/「居中」 切换按钮，控制歌词/字幕文本的水平对齐方式。
  • 状态自动保存，刷新页面后保持上次选择。

🖱️ 滚轮控制

  • 暂停状态下，鼠标滚轮可前后移动播放头，波形随之滚动（播放头居中）。
  • 滚动速度可通过工具栏的「滚速」调节器实时调整（0.1 ~ 5.0 秒）。
  • 播放时滚轮无效，波形自动跟随播放头。

📂 导入与导出

  • 导入：点击「📄 导入歌词/字幕」支持 .txt / .lrc / .srt，或直接拖拽文件到页面。
  • 导出：点击「💾 导出」可选择 TXT（内容跟随当前模式）/ LRC / SRT，调用系统另存为。
  • SRT导出筛选：导出SRT时，可选择「全部导出」「导出单数行」「导出双数行」，方便分离中英文双语字幕。

🎨 其他小功能

  • 最新时间戳：实时显示最后一次更新的时间，清除/删除后自动重算。
  • 深色模式：点击右上角月亮/太阳图标切换，偏好自动保存。
  • 缩放：工具栏「+/-」调节波形显示密度。
  • 悬停游标：鼠标移到波形区域时，显示淡红色竖线预览落点位置。
  • 空格键更新时间戳：在 ⚙️ 配置 中勾选后，按空格键不再控制播放/暂停，而是执行更新时间戳操作。
