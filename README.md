一个简单的LRC/SRT歌词字幕制作工具。如果导入的音乐本身包含很多其他乐器的声音，最好能先把人声提取出来。如果是视频制作SRT字幕，先把视频的音频单独导出来。音频的波形越简单，处理就越简单。
